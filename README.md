## Gokul Saravanan

Product engineer. I build SaaS products, ship them, and keep them running.

10 years across consulting, operations, and full-stack engineering. Started in business strategy (ZS Associates, Swiggy, Foodpanda), then crossed over to writing the software myself. Now I run [Savi](https://savibm.com) and take on fractional CTO work.

### What I've shipped

**[ZestAMC](https://zestamc.com)** — Investment management platform. 7 role-based portals, automated payouts across chains, KYC/AML verification, $10M+ AUM, 200K+ users. Built with React, Hono, Supabase, Redis, Cloudflare.

**[Fenado AI](https://fenado.ai)** — Agentic platform that turns text prompts into working full-stack apps. Real-time generation over WebSocket, 50K+ users. Next.js, LangChain, FastAPI, Firebase.

**[DropTaxi](https://svdroptaxi.com)** — Multi-tenant taxi booking SaaS. One deployment powers every tenant with branded websites, custom domains, and SEO. Astro, Hono, Turso, Drizzle ORM, Fly.io.

**[Frootex](https://frootex.com)** — Custom ecommerce for fresh produce. Location-based delivery zones, real-time inventory sync for perishable goods. Next.js with React Server Components.
